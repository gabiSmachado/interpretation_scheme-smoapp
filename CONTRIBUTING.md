## interpretation_scheme-smoapp Contribution Guidelines

This file is intended to provide some guidance on how to contribute to this project.

## Getting Started

1. Clone this repository and make changes as follows:

```bash
git clone git@github.com:gabiSmachado/interpretation_scheme-smoapp.git
```

2. Set up Python environment and install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the server locally to verify your change:

```bash
python src/server.py
```
